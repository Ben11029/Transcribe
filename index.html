【A.使用说明 Instructions】

1.🔑API 密钥配置：进入设置后，您需要自行获取 API 密钥。针对不同的服务提供商，请查阅其官方文档以获取详细的 API 密钥查询与操作指南。通常情况下，配置 OpenAI 的 API 密钥足以支持系统的基本运行。然而，经过内部测试，我们强烈建议您同时获取 Gemini 的 API 密钥，因为 Gemini 在后续处理方面表现出更高级、更出色的性能。Gemini 提供免费额度 🎁，如需更高级的功能或更大的使用量，您可以绑定支付方式进行升级。

2.🎙️会话稳定性与音频截取：注意保持声源清晰干净，避免多声源。支持麦克风声源，不支持系统内部声源，故一般可以给现场会议或者对外部播放源的转录翻译。根据测试，会话启动后的前一到两段可能会出现不稳定情况。为确保最佳体验，建议您提前截取两段测试音频 🔇，之后会逐渐趋于稳定。在对话过程中，请尽量选择在一段完整、连贯的意群停顿处进行截取。通常建议将每段音频截取为 3 到 4 句话，这样更适合页面显示，提升用户体验 ✨。

3.✍️转录后的编辑与修改：在完成首次转录后，系统会提供编辑修改功能。您可以根据实际听写印象，对关键内容进行修改，以提高转录结果的准确性。请注意，系统目前只提供一次修改机会 ⚠️，一旦您退出对话框或确认修改后，将无法再次进行编辑。在您进行修改操作时，录音将持续进行 🔄，您可以在完成修改工作后，根据时机截取下一段音频。

4.⏯️录音暂停与恢复：在不使用录音功能时，您可以选择暂停录音。暂停后，您可以随时方便地重新开始录音，无需从头设置。
